# Task 2: Data Cleaning & Missing Value Handling

## Objective
To clean the dataset and handle missing values using Python (Pandas, NumPy).

## Dataset Used
- Tehran House Prices Dataset

## Steps Performed
1. Loaded the dataset using Pandas
2. Identified missing values using isnull().sum()
3. Visualized missing values using bar chart
4. Found that Address column had high missing values
5. Removed Address column due to high missing percentage
6. Validated dataset after cleaning
7. Saved cleaned dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Outcome
Improved data quality by handling missing values and preparing a clean dataset.
